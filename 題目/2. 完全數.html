完全數

Description

完全數的定義為某數的所有因數（去除自己）相加等於該數，該數稱做完全數，例如：6的因數有1、2、3與6，去除6後其他的數相加，因為「1+2+3=6」，所以6是第一個完全數。寫一個程式計算出2到給定值的所有完全數。

Input

只有一個正整數。

Output

輸出所以的完全數，有多少個完全數就是多少行。每一個完全數單獨一行。

輸出範例:

6為完全數

Sample Input 1 

10

Sample Output 1

6為完全數
